# Swa2na App 
Android mobile application that present different products from different vendors and enables the authenticated clients to
add/remove products to/from shopping carts and complete the whole online shopping cycle.

# Tools & Libraries
- Android studio.
- Kotlin.
- MVVM.
- Retrofit.
- REST APIS.
- Shopify.
- Firebase Auth.
- Firebase firestore.
- Google pay.
